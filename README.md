# archolith.dev

`archolith.dev` is the public product site for Archolith.

The site currently ships as a static front-end focused on:

- product positioning for the Archolith stack
- the layered "strata" visual system
- brand/logo exploration assets
- screenshots and validation artifacts for the landing page

## Repo layout

- `index.html` — main site entry point
- `hero/` — hero section rendering and slice definitions
- `logos/` — logo variants, showcases, and reference assets
- `screenshots/` — visual verification captures
- `.agent/` — project docs for coding agents

Start with [`.agent/README.md`](./.agent/README.md) for project-specific architecture, data-model, and workflow guidance.
